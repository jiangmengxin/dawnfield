## 当前高优先级体验债

- 三选一按钮高度、间距和长文案自适应。
- HUD 尺寸体系与令牌等比缩放。
- 规则卡覆盖层矮屏滚动/分页。
- 角色卡和地图卡结构区分。
- 标签页、按钮、Toast 的组件收敛。
- 屏幕震动和动效强度统一。
